## Project Description

This project uses a movie dataset containing overview text and numeric attributes to build machine learning models. The project involves data cleaning, feature engineering, model training, evaluation, and TinyML deployment. We also completed the Bonus tasks which include visualizations and outlier analysis. The goal is to handle mixed-type data and gain insights from combined text and numeric features.
